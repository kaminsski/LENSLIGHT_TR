# Lenslight_TR
 Lenslight_TR website is built with the NodeJS. 
This website is a photo gallery. Allows users to share photos and see recently shared photos



## Functions/features of the website
* Login/Register
* Create/Update/Delete Photo
* Gallery


## Technologies used


* HTML5
* CSS3
* JavaScript (ES6)
* GitHub
* Bootstrap
* NodeJS
* EJS
* MongoDB
* ExpressJS
